Bootstrap (також відомий як Twitter Bootstrap) - вільний набір інструментів для створення сайтів і веб-додатків. Містить HTML і CSS шаблони оформлення для типографіки, Web-форм, кнопок, міток, блоків навігації та інших компонентів Web-інтерфейсу, включно з JavaScript-розширеннями.

Bootstrap використовує найсучасніші напрацювання в галузі CSS і HTML, тому необхідно бути уважним при підтримці старих браузерів.
